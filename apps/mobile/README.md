# Renova Mobile App

Aplicativo móvel desenvolvido com Expo e React Native.

## Scripts

- `pnpm dev --filter @renova/mobile` - Inicia o servidor de desenvolvimento do Expo.
- `pnpm lint --filter @renova/mobile` - Executa o linting do projeto.
- `pnpm test --filter @renova/mobile` - Executa os testes unitários.
